# EatNVibe API

## Installation

```bash
# install dependencies
yarn install
```

## Running the app

### Start Docker and run the following command

```bash
# start docker
docker-compose up
```

### Running the app

```bash
# development
yarn start

# watch mode
yarn start:dev

# production mode
yarn start:prod
```

## Test

```bash
# unit tests
yarn test

# test coverage
yarn test:cov
```
